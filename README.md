# Dogs API

This is an Express/Postgresql RESTful API. It uses Babel for modern syntax.

# Install

Clone this repository, and run:

`npm install`

After that, make a `.env` file in the root directory, and add your database name, username, and password, as well as your express port:

```
PORT=3000
DATABASE=dogs_api_development
DATABASE_USERNAME=postgres
DATABASE_PASSWORD=password
```
Then, run `npm start` and you should be good to go!
